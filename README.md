# ⌚ Watch Store Management System (Laravel)

### 🌟 Overview
A comprehensive E-commerce platform for a luxury watch store, built using the **Laravel** framework. The project features a seamless shopping experience for customers and a robust Admin Dashboard for managing inventory and orders.

### 🚀 Live Demo
**URL:** [https://mariam-watch-store.onrender.com](https://mariam-watch-store.onrender.com)
*(Note: As the project is hosted on a free tier, the initial load may take a few seconds while the server wakes up).*

### 🛠 Tech Stack
* **Backend:** PHP 8.x & Laravel 10/11
* **Database:** MySQL (Hosted on Aiven Cloud)
* **Frontend:** Blade Templates, CSS, JavaScript, Vite
* **Deployment:** Render (Web Service)

### 💎 Key Features
* **Customer Interface:**
    * Browse premium watch collections and categories.
    * Detailed product view pages.
    * Fully functional Shopping Cart (Add, Update, Remove).
    * Secure Checkout and order placement process.
* **Admin Dashboard:**
    * Complete **CRUD** operations (Add, Edit, Delete watches).
    * Order management system to track and review customer purchases.
    * Secure Admin Authentication system.

### 🔧 Technical Implementation
During the development and deployment phase, the following tasks were successfully completed:
1. **Cloud Database Integration:** Connected the application to a remote MySQL database on Aiven Cloud with SSL encryption.
2. **Environment Configuration:** Managed sensitive credentials and app keys using secure Environment Variables on Render.
3. **CI/CD Workflow:** Established a deployment pipeline from GitHub to Render for automatic updates.
4. **Responsive UI:** Ensured the storefront is fully responsive and compatible with mobile and desktop devices.
